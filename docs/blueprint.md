# **App Name**: Temporal Atlas

## Core Features:

- Interactive World Map: Display an interactive world map with temperature gradients overlaid to represent regional temperatures.
- Temperature Visualization: Visualize temperature data using a color-coded gradient on the map, as shown in the attached reference image.
- Yearly Temperature Simulation: Ability to scroll through different years (e.g., 2023, 2024, 2025) to observe changes in regional temperatures over time using mock data for testing.
- Data Interpolation: Smooth temperature transitions between regions based on available data points.
- Dynamic Text Generation: AI tool to generate a short descriptive analysis for a given year's data. This text will be displayed on the page as context.

## Style Guidelines:

- Primary color: Sky blue (#66B2FF) to represent the coolness associated with low temperatures.
- Background color: Light gray (#F0F0F0) to provide a neutral backdrop.
- Accent color: Coral red (#FF7F50) to highlight warmer temperature regions.
- Body and headline font: 'Inter', a sans-serif font, for clean and modern readability.
- A clean and intuitive layout with the map as the central focus, accompanied by a timeline slider for year selection and the temperature scale at the bottom.
- Simple, clear icons for interactive elements, such as the year selector and zoom controls.
